# EduBoom CRM - Kevin’s Academy Demo Account

Open `eduboom-crm-kevins-demo.html` in a browser.

This version is preloaded with a realistic Kevin’s Academy demo account:

- 150 students
- Courses: ESL General English, IELTS Preparation, CEFR Preparation, Speaking Club
- 12 groups
- 8 teachers
- 96 leads with funnel statuses
- realistic payments, debts, attendance and teacher salary data

The app saves changes in browser localStorage under a separate demo key.
For real multi-user daily use, connect this UI to a database and authentication.
